# Sample Web

A clean, responsive static website built using HTML, CSS, and JavaScript. This repository serves as a foundation for a multi-page website, complete with styling, interactive elements, and routing for a custom domain.

## 📂 Project Structure

This project follows a standard static web architecture:

* **`index.html`**: The main landing/home page of the website.
* **`about.html`**: Contains information about the site, author, or business.
* **`projects.html`**: A showcase of portfolio items or past work.
* **`contact.html`**: A page with contact information or a reach-out form.
* **`css/`**: Directory containing all cascading stylesheets for styling the website.
* **`js/`**: Directory containing JavaScript files for interactive frontend features.
* **`img/`**: Directory for storing image assets used across the site.
* **`CNAME`**: Configuration file for hosting the site on a custom domain (often used with GitHub Pages).
